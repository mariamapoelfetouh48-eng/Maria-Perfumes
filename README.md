# 🌸 MARIA | Luxury Fragrance Experience

A premium, high-end landing page design project for a luxury fragrance brand. This project emphasizes minimal design, typography, and user experience.

---

## ✨ Key Features

* **Premium Design Identity:** Built with a color scheme of "Rose Gold & Deep Brown" to give it a luxury feel.
* **Responsive Design:** Optimized for ultra-wide screens as well as mobile phones.
* **Interactive UI Elements:**
	+ **Smooth Dropdowns:** Clean, bullet-free dropdown menus.
	+ **Elegant Pricing:** Interactive pricing cards that display details when hovered.
	+ **Minimalist Team Section:** Elegant team member profiles with a zoom effect.
	+ **Professional Footer Section:** A well-structured footer section with a newsletter and quick links.

---

## 🛠️ Built With (Tech Stack)

* **HTML5:** Semantic markup for better search engine optimization and accessibility.
* **CSS3:** Utilized for advanced Flexbox/Grid layouts and custom keyframe animations.
* **Font Awesome:** High-quality scalable vector graphics library.
* **Google Fonts:** *Playfair Display* used for headings, *Poppins* used for body copy.
